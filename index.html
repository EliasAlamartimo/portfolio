Projects

2D swing game

I made this simple 2D platformer with a swinging mechanic as my first game.

This project was made to learn the basics of the Unity game engine and C#.

Conqueror of the jungle

A 3D doom inspired first person shooter that I made paired with a classmate.

This project was used to enhance our proficiency in the Unity game engine and to learn some teamwork skills.

This game had multiple mechanics such as weapon and character upgrades and enemies that chase the player.

Also in this project we utilized version control for the first time.

Cosmic catastrophe

This game was made in a team of 6, with me being the gameplay programmer.

In this game you need to catch meteorites using your hands, which are tracked using mediapipe library.

This game was made for us to learn how to use python and its libraries, such as pygame and mediapipe.

In this project we had an even bigger team to further enhance our teamwork skills.
